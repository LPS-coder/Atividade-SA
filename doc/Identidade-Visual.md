**Estudo da Identidade Visual – Sistema Ferroviário OMNIRAIL**

A identidade visual do sistema ferroviário OMNIRAIL foi criada para transmitir tecnologia, inovação, monitoramento inteligente e segurança operacional. Inspirado no personagem Omni-Man, da série Invencível, o nome OMNIRAIL representa força, controle e alta performance no gerenciamento ferroviário.

O sistema possui uma interface moderna e intuitiva, voltada para monitoramento em tempo real, análise de desempenho ferroviário e gestão inteligente de sensores IoT.

**Cores**

As principais cores utilizadas são:

Vermelho (#E30613): representa energia, alerta, tecnologia e impacto visual;
Branco (#FFFFFF): utilizado para limpeza visual e melhor legibilidade;
Cinza claro: aplicado no fundo para conforto visual e organização;
Verde, amarelo e vermelho: usados para representar os status operacionais do sistema.
Tipografia

Foi utilizada uma fonte sem serifa, moderna e tecnológica, proporcionando melhor leitura em dashboards, tabelas e relatórios operacionais.

Os títulos possuem destaque em negrito para facilitar a navegação e organização das informações.

**Estilo Visual**

O sistema possui um estilo:

Moderno;
Tecnológico;
Corporativo;
Minimalista;
Funcional.

Os componentes utilizam cards, gráficos, ícones minimalistas e bordas arredondadas, criando uma experiência visual profissional e organizada.

**Referências Visuais**

A interface foi inspirada em:

Sistemas de monitoramento ferroviário inteligente;
Dashboards corporativos;
Plataformas de mobilidade urbana;
Sistemas IoT em tempo real;
Painéis administrativos modernos.

Os ícones de trem, sensores, gráficos, alertas e mapas reforçam a proposta tecnológica da plataforma.

**Proposta de Layout**

O layout foi desenvolvido para facilitar o monitoramento operacional e a tomada de decisões em tempo real.

**Tela de Login**

Possui:

Campos de e-mail e senha;
Botão “Entrar”;
Card centralizado;
Design moderno com detalhes em vermelho.

O sistema valida as credenciais do usuário antes de permitir o acesso ao dashboard administrativo.

Dashboard Principal

**Exibe:**

Nome do administrador autenticado;
Botões de navegação rápida;
Indicadores operacionais;
Lista de sensores cadastrados.

**Funções disponíveis:**

Cadastro de Sensores e Trens;
Monitoramento em Tempo Real;
Relatórios;
Logout.
Tela de Cadastro de Sensores e Trens

**Permite cadastrar:**

Nome do sensor;
Localização;
Tipo de dado monitorado;
Trem vinculado.

Os dados são armazenados no banco de dados para monitoramento da malha ferroviária.

Tela de Monitoramento em Tempo Real

**Exibe:**

Velocidade do trem;
Localização em mapa interativo;
Status operacional;
Consumo de energia;
Histórico de falhas;
Gráficos de desempenho.

Os dados são simulados em tempo real utilizando sensores IoT.

**Tela de **Relatórios**

Permite:

Gerar relatórios;
Filtrar por período;
Filtrar por tipo de falha;
Visualizar gráficos analíticos;
Acompanhar tendências operacionais.
Objetivo

**A plataforma OMNIRAIL busca oferecer uma solução moderna, inteligente e eficiente para monitoramento ferroviário em tempo real, auxiliando no controle operacional, manutenção preditiva e segurança da ferrovia.**
